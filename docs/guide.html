Plane

This mode performs standard domain coloring: every point on the complex plane is passed through a function. The argument of the result determines the hue at that point, and the norm of the result determines the value (lightness/darkness).
Tip: plotting the identity function shows the current color map.

Riemann Sphere

This mode performs domain coloring on the Riemann Sphere. See here for more details.

3D Mode

This mode plots the norm of a complex function on the \( z \)-axis (vertical axis). The coloring at each point works the same as plane mode.
Tip: The norms of the outputs of complex functions can get big very quickly and change rapidly. Using \( \operatorname{squeeze} \) can help control the norms to make things easier to visualize. Recommended usage to plot the function \( f(z) \): \( \operatorname{squeeze}(f(z), 0.99, 10) \).
